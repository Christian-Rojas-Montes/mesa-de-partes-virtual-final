# Hoja de ruta de cambios

El desarrollo futuro seguirá este orden y cada punto se ejecutará como una fase independiente:

1. Modelo configurable de trámites y requisitos.
2. Administración de catálogos.
3. Formulario dinámico para solicitantes.
4. Modalidades digital, híbrida y presencial.
5. Registro de trámite físico.
6. Historial y notificaciones.
7. Convalidaciones.
8. Titulación.
9. Revisión de seguridad.
10. Rediseño institucional.
11. Preparación para despliegue.
12. Pruebas integrales.
13. Actualización final del documento EFSRT.

No se debe adelantar una fase ni implementar funcionalidades de fases posteriores sin solicitud expresa. Los cambios funcionales y estéticos se mantendrán separados.
