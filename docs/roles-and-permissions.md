# Roles y permisos

| Rol | Alcance principal |
| --- | --- |
| Solicitante | Consulta catálogo y administra únicamente sus solicitudes, documentos, citas y notificaciones. |
| Mesa de Partes | Registra, revisa, observa, recibe y deriva expedientes documentarios autorizados. |
| Responsable de área | Atiende expedientes asignados exclusivamente a su área. |
| Administrador | Gestiona usuarios, catálogos, parámetros y consulta general autorizada. |

El personal académico solo accede cuando existe un rol y autorización explícitos. Ocultar botones no sustituye las Policies ni el middleware. Toda consulta, exportación, descarga o modificación debe aplicar el mismo alcance en servidor. Las acciones sensibles y exportaciones se auditan.
