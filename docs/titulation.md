# Titulación

El módulo extiende las solicitudes documentarias sin convertir la aplicación en un sistema académico. Conserva solicitud principal, modalidad, etapa, convocatoria, programación, resultado, documentos, responsables e historial.

## Modalidades

- Trabajo de Aplicación Profesional: propuesta, aprobación, constancia de egresado, expediente de sustentación, programación y resultado. Límites de integrantes, plazos, similitud y nota son configurables.
- Examen de Suficiencia Profesional: experiencia/EFSRT, requisitos, programación, oportunidades y resultado teórico-práctico con parámetros configurables.

Después de una aprobación se habilita el expediente final de título, con requisitos digitales y físicos, conformidad, subsanación, registro, emisión, recojo y entrega. Actas, DNI, partida, fotografías y documentos académicos son privados. No se implementan motor antiplagio, banco de preguntas ni cálculos académicos complejos.
