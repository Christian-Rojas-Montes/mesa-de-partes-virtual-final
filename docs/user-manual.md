# Manual del usuario solicitante

1. Consulte **Trámites** y use búsqueda o filtros para revisar requisitos, modalidad, costo referencial y disponibilidad.
2. Pulse **Iniciar trámite**. Si no inició sesión, autentíquese; el sistema conservará la selección.
3. Elija la variante, complete la elegibilidad y los campos solicitados.
4. Adjunte solo documentos requeridos. Los requisitos exclusivamente físicos se presentan después, según instrucciones.
5. Revise el resumen y confirme. Guarde el código de seguimiento.
6. Desde el panel consulte estado, línea de tiempo, observaciones, faltantes, citas, originales pendientes y documento listo.
7. Responda observaciones dentro del plazo y marque notificaciones como leídas.

Para trámites híbridos o presenciales, la declaración del usuario no equivale a recepción: Mesa de Partes debe confirmarla. No comparta su cuenta ni cargue documentos de otras personas.
